CSS3 border-radius support

The border-radius CSS3 property allows elements to have rounded corners. Of commonly encountered browsers, only IE6/7/8 do not support this property. The shim by CSS3Pie allows them to display rounded corners. On IE6/7/8, the elements being rounded need to be positioned or floated in order for the background to show correctly.
