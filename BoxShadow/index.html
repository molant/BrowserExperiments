CSS3 box-shadow support

The box-shadow CSS3 property allows elements to have a shadow, giving them the illusion of "floating" over the page. Of commonly encountered browsers, only IE6/7/8 do not support this property. The shim by CSS3Pie allows them to display a box shadow. On IE6/7/8, the elements need to be positioned or floated in order for the background to show correctly. Note that rendering of shadows and default values can vary across browsers, so all values should be specified and the result should be visually confirmed in each browser.
