CSS3 linear-gradient support

The (w3c) CSS3 linear-gradient lets us add gradients as background images to block elements. This shim by CSS3Pie allows IE6/7/8/9 to display a linear-gradient; other browsers require vendor-specific CSS background properties. On IE6/7/8/9, the elements need to be positioned or floated in order for the background to show correctly.
