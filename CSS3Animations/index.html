jQuery CSS Animation support (from jQuery Animate Enhanced by Ben Barnett)

On platforms such as the iPhone and iPad, the hardware-enhanced CSS3 Transitions perform much more smoothly than JavaScript animations. This jQuery plugin modifies the standard jQuery .animate() method to use CSS3 Transitions when they are available. Animations that use the left, top, and/or opacity properties may benefit. (CSS3 Transitions specification)

NOTE: This browser does not support CSS Transitions; it will use the standard jQuery JavaScript animations.
