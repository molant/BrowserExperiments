Opacity support

IE 6/7/8 do not support the CSS opacity property, but they do support giving opacity to an element via a filter. By specifying both opacity and filter in a stylesheet, it is possible to support all browsers. The jQuery framework supports the opacity property in all browsers in its .css() method, so no special-case code is needed to accomodate opacity in most scripts.

To implement opacity shims without jQuery, see the Microsoft IE Blog.
