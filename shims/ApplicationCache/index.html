Application Cache

The (w3c) Application Cache specifies a mechanism for selecting files to store and load from the clients computer. Currently this is only supported by a handful of browsers and FF3.6 even broke their implementation so adoption has been thwarted.

The shim HTML5-gears by Brad Neuberg attempts to add support for this in IE browsers (although i have failed to get the shim working on IE myself, perhaps my computer is special and you'll have better luck)

If you disable internet connection this line should still be yellow, from the linked in stylesheet, and the image should be displayed. However the site style should now be absent.
