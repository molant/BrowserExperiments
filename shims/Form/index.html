Input Types

The (w3c) HTML5 form  input types and attributes have made a spec for some very common features you see in html forms, such as a date picker or a autocomplete. This is good news as it allows devices such as mobile phones to interpret the control natively, and for example take advantage of the devices date picker.

The shim is part of the Webshim project by afarkas and shims up all current browsers, at present none include a full implementation.
