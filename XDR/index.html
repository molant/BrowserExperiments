XDomainRequest support (from jQuery.ajax hooks)

The XDomainRequest object (XDR) allows Internet Explorer 8 and higher to make a limited set of cross-domain AJAX requests, per the W3C Cross Origin Resource Sharing (CORS) specification. This shim (specifically, a jQuery transport) allows jQuery's $.ajax() method to make XDR requests in IE8 and IE9. There is no shim for older versions of IE. (Microsoft XDomainRequest documentation) (Microsoft XDR limitations and workarounds)

NOTE: This browser does not support XDomainRequest; it may support CORS through XMLHTTPRequest. The demo will try using both via $.ajax().
